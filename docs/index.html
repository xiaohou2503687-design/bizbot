FAQ

How is this different from Intercom/Crisp?

Intercom starts at $74/mo. BizBot is self-hosted, open source, and you own your data. One script tag vs. a platform you're locked into.

Do I need an OpenAI API key?

No! BizBot works with a local keyword search engine by default. Add your OpenAI key for enhanced AI responses — your choice.

Can I host it on my own server?

Yes. BizBot is a Node.js server you run anywhere — VPS, Railway, Render. Full control, no vendor lock-in.
