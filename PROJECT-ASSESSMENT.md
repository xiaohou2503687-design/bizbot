## 📊 立体评估

### 1. 市场维度 ⭐⭐⭐⭐⭐ (5/5)

| 指标 | 数据 |
|------|------|
| **TAM** | AI 客服市场 $15B+，2026 年爆发增长 |
| **SAM** | 独立开发者产品 + 微SaaS ~2M |
| **SOM** | 第一年 1K-10K 用户 |
| **竞争格局** | Intercom($74/月起)、Crisp(免费版很弱)、Tidio(中端)、自建RAG(技术要求高) |
| **市场缺口** | 没有"零配置、复制粘贴一个 script 就完事"的独立开发者客服工具 |

**关键信号:**
- 每个独立开发者都在 Twitter 抱怨"回复支持邮件占用太多时间"
- ChatGPT 让用户对 AI 客服接受度飙升
- RAG(检索增强生成)技术成熟，准确率 95%+
- Intercom 太贵 → 独立开发者用不起 → 巨大空白

### 2. 技术维度 ⭐⭐⭐⭐ (4/5)

| 维度 | 评估 |
|------|------|
| **复杂度** | 中 — RAG + 嵌入式 Widget + 多渠道 |
| **核心依赖** | Embedding API、向量数据库(LanceDB/Chroma)、WebSocket |
| **数据存储** | 向量库+SQLite |
| **AI 层** | RAG 问答、意图识别、升级人工判断 |
| **估时** | 2周 MVP，4周完整版 |
| **技术风险** | 🟡 中 — 幻觉控制、多轮对话质量 |

### 3. 营收维度 ⭐⭐⭐⭐⭐ (5/5)

| 定价方案 | 月价 | 目标用户 |
|----------|------|----------|
| **Free** | $0 | 每月 50 次对话，基础 RAG |
| **Pro** | $19/月 | 500 次对话，自定义品牌+邮件摘要 |
| **Business** | $49/月 | 3K 次对话，多渠道(邮件+Widget+Slack) |
| **Lifetime** | $199 | 一次性买断+自带 API Key |

**首年收入预测:**
- 保守: 400 用户 × $19 = $7,600/月 → $91K/年
- 中等: 1K 用户 × $25(混合) = $25,000/月 → $300K/年
- 乐观: 3K 用户 → $500K+/年

**货币化策略:**
- 对话量大 = 用得多 = 自然升级 → 完美的 PLG 模型
- "你的用户问的 80% 问题都一样" → 价值主张极其清晰
- 与 ShipFast 深度集成 → 部署完自动推荐接入

### 4. 风险维度 ⭐⭐⭐ (3/5)

| 风险 | 等级 | 缓解 |
|------|------|------|

| 大厂竞争(OpenAI/Google) | 🟡 中 | 聚焦独立开发者 Niche |
| 用户不信任 AI | 🟡 中 | 无缝升级至人工+对话评分系统 |
| API 成本 | 🟡 中 | 支持自带 API Key，本地模型选项 |
